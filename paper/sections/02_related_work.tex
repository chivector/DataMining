\section{相关工作}

\paragraph{大语言模型评测。}
GPT-3 的少样本学习结果表明，大规模语言模型可以通过纯文本提示完成广泛任务 \cite{brown2020language}。随后，MMLU 和 BIG-bench 等基准进一步将评测范围拓展至多学科知识、常识、数学和复杂任务集合 \cite{hendrycks2021measuring,srivastava2022beyond}。这些工作强调了跨任务能力测量的重要性，然而平均分数往往难以揭示模型在特定条件下的失效根因。本文选取一个边界更明确的空间推理场景，通过受控变量和重复采样刻画模型行为差异。

\paragraph{推理提示与行为测试。}
Chain-of-thought prompting 表明，中间推理步骤可以显著改变模型在复杂推理任务上的表现 \cite{wei2022chain}。然而可解释的推理文本并不必然意味着稳定的推理过程，尤其当题干中出现无关信息或近阈值参数时，模型可能生成看似合理但结论错误的解释。CheckList 提出从行为测试角度构造能力矩阵和测试类型，以避免仅凭总体准确率评测 \cite{ribeiro-etal-2020-beyond}；对抗阅读理解工作亦表明，无关或误导性文本可以显著破坏模型判断 \cite{jia-liang-2017-adversarial}。本文借鉴上述行为测试思想，将扰动、层级和反事实条件直接纳入 Prompt 设计。

\paragraph{空间推理。}
空间关系是人类认知和语言理解的基础。SPARTQA 基准系统测量了模型对空间问答中方位介词的推断能力，揭示出大语言模型在空间关系推断上与人类表现存在明显差距 \cite{mirzaee2021spartqa}；后续基准亦发现 LLM 在空间介词推断上存在显著的提示敏感性 \cite{comsa-narayanan-2023-benchmark}。与上述工作不同，本文聚焦一个纯文本、参数化、可判定的轮椅转弯任务：模型无需识别图像，而须从自然语言描述中提取几何约束，并给出通行判断。这一设计适合在现实工程语境中观察语言模型的空间数量推理边界，而非泛化的空间介词能力。

\paragraph{自一致性、幻觉与推理稳定性。}
Wang 等人提出的自一致性解码策略通过对同一问题进行多次采样并取多数投票来提升推理准确率\cite{wang2023self}，表明重复采样本身携带可利用的诊断信号。然而高自一致性并不保证高准确率：Turpin 等人发现，模型生成的链式推理文本常与其实际推理过程相悖，形成所谓"不忠实的 CoT"\cite{turpin2023language}；Ji 等人的幻觉综述进一步表明，LLM 在信息不足场景中最常见的失效形式并非随机错误，而是以高置信度填充虚假内容\cite{ji2023survey}。本文的"高自一致性但低准确率"现象与上述文献高度契合，并提供了空间几何约束场景下的系统性实证。

\paragraph{数据挖掘方法与行为分析。}
关联规则挖掘由 Agrawal 等人奠定形式化基础\cite{agrawal1993mining}；Han 等人提出的 FP-Growth 算法通过构建频繁模式树消除候选集生成开销，成为大规模离散事务挖掘的标准方法\cite{han2000mining}。基于 scikit-learn\cite{pedregosa2011scikit} 的 K-Means 聚类与决策树在机器学习领域被广泛用于行为分组与可解释特征提取。本文将上述算法应用于 LLM 回复特征表，使挖掘结果可直接回溯至原始文本，与 HELM 等大规模排名评测\cite{liang2022holistic}形成互补——前者关注同一任务内的失效解释，后者关注跨任务的能力比较。
